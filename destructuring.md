# Array Destructuring

## Basic example

- instead of:

```js
const numbers = [10, 20, 30];
const first = numbers[0];
const second = numbers[1];
console.log(first, second); // 10, 20
```

- You can use array destructuring

```js
const [first, second] = numbers;
console.log(first, second); // 10, 20
```
